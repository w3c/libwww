This version of the stream object just writes to a socket. The socket is assumed open and closed afterward.There are two versions (identical on ASCII machines) one of which converts to ASCII on output.

This module is implemented by HTWriter.c, and it is a part of the Library of Common Code.

Bugs:

  • strings written must be less than buffer size. 
#ifndef HTWRITE_H
#define HTWRITE_H

#include "HTStream.h"

extern HTStream * HTWriter_new		PARAMS((SOCKFD soc));
extern HTStream * HTWriter_newNoClose	PARAMS((SOCKFD soc));

extern HTStream * HTASCIIWriter		PARAMS((SOCKFD soc));

#endif
end
